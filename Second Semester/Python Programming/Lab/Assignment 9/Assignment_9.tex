\subsection{Metacharacters and Special Sequences}

\subsubsection{Metacharacters}
Metacharacters are characters with a special meaning:
\begin{enumerate}
    \item \textbf{\^{}} - Starts with
    \item  \textbf{\$} - Ends with
    \item  \textbf{.} - Any character
    \item  \textbf{|} - Either or
    \item  \textbf{[]} - Range
    \item  \textbf{()} - Capture and group
    \item  \textbf{\textbackslash} - Escape character
    \item  \textbf{\textbackslash d} - Digits
    \item  \textbf{\textbackslash D} - Not a digit 
    \item  \textbf{\textbackslash w} - Word character
    \item  \textbf{\textbackslash W} - Not a word character
    \item  \textbf{\textbackslash s} - Whitespace
    \item  \textbf{\textbackslash S} - Not a whitespace
    \item  \textbf{\textbackslash b} - Word boundary
    \item  \textbf{\textbackslash B} - Not a word boundary
    \item  \textbf{\textbackslash A} - Beginning of string
    \item  \textbf{\textbackslash Z} - End of string
    \item  \textbf{\textbackslash z} - End of string
    \item  \textbf{\textbackslash G} - End of previous match
    \item  \textbf{\textbackslash n} - New line
    \item  \textbf{\textbackslash t} - Tab
    \item  \textbf{\textbackslash} - Backslash
    \item  \textbf{\textbackslash (escape)} - Escape character
    \item  \textbf{\textbackslash \{number\}} - Occurrences
    \item  \textbf{\textbackslash \{number, number\}} - Range of occurrences
    \item  \textbf{\textbackslash \{number, \}} - Minimum occurrences
    \item  \textbf{\textbackslash * } - Zero or more occurrences
    \item  \textbf{\textbackslash +} - One or more occurrences
    \item  \textbf{\textbackslash ?} - Zero or one occurrences
\end{enumerate}

They are used to define the search pattern. They are used in the search() methods to search a string for a match.

\subsection{Regular Expression Functions}

\subsubsection{re.findall()}

The re.findall() method returns a list of strings containing all matches. If there are no matches, an empty list is returned. 

Parameters: This method takes following arguments :

\begin{itemize}
    \item pattern: A regular expression to match the string.
    \item string: A string to be searched.
    \item flags: We can specify different flags using bitwise OR (|). These are modifiers, which are listed in the table below.
\end{itemize}

\begin{lstlisting}[language=python]
    pattern = r"eggs"
    string = "bacon and eggs"
    re.findall(pattern, string, flags=0)
    # output = ['eggs']
\end{lstlisting}

\subsubsection{re.split()}

The re.split method splits the string where there is a match and returns a list of strings where the splits have occurred.

Parameters: This method takes following arguments :

\begin{itemize}
    \item pattern: A regular expression to match the string.
    \item string: A string to be split on basis of pattern matching.
    \item maxsplit: It is a number, which tells us to split the string into maximum of provided number of times. If not provided, the default is -1 that is, all occurrences.
    \item flags: It is an optional parameter that can be used to control different regular expression operations.
\end{itemize}

\begin{lstlisting}[language=python]
    pattern = r"\s"
    string = "bacon and eggs"
    re.split(pattern, string, maxsplit=0, flags=0)
    # output = ['bacon', 'and', 'eggs']

    re.split(pattern, string, maxsplit=1, flags=0)

    # output = ['bacon', 'and eggs']
\end{lstlisting}


\subsubsection{re.sub()}

The re.sub() method replaces all occurrences of the RE pattern in string with repl, substituting all occurrences unless max provided. This method returns modified string.

Parameters: This method takes following arguments :
\begin{itemize}
    \item pattern: A regular expression to match the string.
    \item  repl: A substitute of the replace string.
    \item  string: A string to be matched.
    \item  count: Maximum number of occurrences that need to be replaced.
    \item  flags: It is an optional parameter that can be used to control different regular expression operations.
\end{itemize}
\begin{lstlisting}[language=python]
    pattern = r"eggs"
    string = "bacon and eggs"
    repl = "spam"
    re.sub(pattern, repl, string, count=0, flags=0)
    # output = 'bacon and spam'
\end{lstlisting}

\subsubsection{re.search()}

The re.search() method takes two arguments: a pattern and a string. The method looks for the first location where the RegEx pattern produces a match with the string.

If the search is successful, re.search() returns a match object; if not, it returns None.

Parameters: This method takes following arguments :

\begin{itemize}
    \item pattern: A regular expression to match the string.
    \item string: A string to be searched.
    \item flags: We can specify different flags using bitwise OR (|). These are modifiers, which are listed in the table below.    
\end{itemize}


\begin{lstlisting}[language=python]
    pattern = r"eggs"
    string = "bacon and eggs"
    re.search(pattern, string, flags=0)
    
    # output = <re.Match object; span=(10, 14), match='eggs'>

    pattern = r"bacon"

    re.search(pattern, string, flags=0)
    # output = <re.Match object; span=(0, 5), match='bacon'>

    pattern = r"sausage"
    re.search(pattern, string, flags=0)

    # output = None

\end{lstlisting}


\subsubsection{re.match()}

The re.match() function returns a match object on success, None on failure. We use group(num) or groups() function of match object to get matched expression.

Parameters: This method takes a regular expression pattern and a string and searches for that pattern with the string.

\begin{lstlisting}[language=python]
    pattern = r"eggs"
    string = "bacon and eggs"
    re.match(pattern, string, flags=0)
    # output = None

    pattern = r"bacon"
    re.match(pattern, string, flags=0)
    # output = <re.Match object; span=(0, 5), match='bacon'>

    pattern = r"sausage"

    re.match(pattern, string, flags=0)
    # output = None
\end{lstlisting}

\section{FAQ}
\begin{enumerate}
    \item \textbf{Write a Python program to search some literals strings in a string. Sample text: 'The quick brown fox jumps over the lazy dog.' -- Searched words: 'fox', 'dog', 'horse'.}

\begin{lstlisting}[language=python]
import re
text = 'The quick brown fox jumps over the lazy dog.'
print(re.findall(r'fox|dog|horse', text))
# Output: ['fox', 'dog']
\end{lstlisting}
    \item \textbf{Explain the findall () and sub () methods?}
    The re.findall() method returns a list of strings containing all matches. The Parameters used in this function are: 
    \begin{itemize}
        \item pattern which is the regular expression to be matched.
        \item string which is the string where pattern would be searched to fetch the matching strings.
        \item flags which are optional and can be used to modify the meaning of the given pattern.
    
    \end{itemize}

    The re.sub() method returns the string obtained after replacing or substituting the specified patterns in the string. The Parameters used in this function are:

    \begin{itemize}
        \item pattern which is the regular expression to be matched.
        \item repl which is the substitute for the matched string.
        \item string which is the string where pattern would be searched to fetch the matching strings.
        \item count which is optional and can be used to specify the number of occurrences to be replaced.
        
    \end{itemize}

    Here's an example of findall() and sub() methods:

\begin{lstlisting}[language=python]
import re
text = 'The quick brown fox jumps over the lazy dog.'
print(re.findall(r'fox|dog|horse', text))
# Output: ['fox', 'dog']
print(re.sub(r'fox|dog', 'cat', text))
# Output: The quick brown cat jumps over the lazy cat.
\end{lstlisting}

\end{enumerate}
